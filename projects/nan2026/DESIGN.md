# NAN 2026 게임 설계 — 한 판 스펙 v0.1

> 제목: **미정** (가제: *TETRA* / *사원소 편대*). 이 문서는 살아있는 설계 스펙이며, 진행하며 갱신합니다.
> 시간순 논의·결정 기록은 [log.md](log.md) 참고.

## 한 줄 컨셉
**뱀서식 무기 아스널 × 물·불·풀·노말 타입 상성 × 비행 슈팅.** 랜덤 테마 스테이지를 돌며 타입을 맞춰 갖춰 양학하는 ~30분 로그라이트.

## 설계 기둥 (Design Pillars)
- **능동 공격** — 자동발사, 유저는 이동·위치만 (194X식). 여러 무기 동시 발사(뱀서식).
- **선택이 만든 양학** — 타입 커버리지 + 무기 맥스 투자로 얻어내는 지배.
- **정보 기반 선택 > 운빨** — 테마 스테이지로 적 타입을 예측 → 판단이 운을 이김.
- **적당한 관대 숙련** — 느리고 큰 텔레그래프 탄막, 무-트위치. 압박은 반응이 아니라 위치·판단.
- **의미 있는 이동** — 알맞은 무기를 알맞은 적/부위에 대는 위치잡기. 바운드 아레나, 무-노가다(자동수집).
- **무API·오프라인·저용량 바닐라** — AI는 빌드타임에 콘텐츠 생성 + 시뮬 밸런싱.

## 타입 상성 (4종)
삼각 상성 + 노말(무상성 안정딜).

| 공격 ↓ \ 방어 → | 물 | 불 | 풀 | 노말 |
|---|---|---|---|---|
| **물** | ×1 | **×2** | ×0.5 | ×1 |
| **불** | ×0.5 | ×1 | **×2** | ×1 |
| **풀** | **×2** | ×0.5 | ×1 | ×1 |
| **노말** | ×1 | ×1 | ×1 | ×1 |

- 물 > 불 > 풀 > 물 (효과적 ×2 / 역상성 ×0.5)
- **노말**: 공격이든 방어든 항상 ×1 → *상성 안 타는 안정적 축*. 노말 무기 = 꾸준한 기본딜, 노말 적 = 약점 없는 탱커.

## 무기 (2축: 거동 × 타입) + 뱀서식 진화
무기 = **거동**(어떻게 나가나) × **타입**(뭐에 강한가). 각 무기 Lv1~8. 저레벨은 약하지만 **맥스까지 키우면 상위 폼으로 진화 + 효과 폭발**(투자·인내의 보상 = 얻은 양학).

거동 종류: **유도 / 일자 관통 / 방사형 / 궤도(오빗) / 설치**

예시 무기(초안):
| 무기 | 거동 | 타입 | 맥스 진화 |
|---|---|---|---|
| 아쿠아 시커 | 유도 | 물 | 분열 유도 어뢰 |
| 플레임 랜스 | 일자 관통 | 불 | 화면 관통 화염방사 |
| 브램블 버스트 | 방사형 | 풀 | 전방위 가시밭 |
| 타이드 마인 | 설치 | 물 | 광역 소용돌이 |
| 위스프 | 유도 | 불 | 다중 추적 도깨비불 |
| 오빗 드론 | 궤도 | 노말 | 드론 3기 + 요격 |

- **빌드 텐션**: 넓게 퍼뜨려 타입 커버리지(안전) vs 하나 몰빵 맥스(초반 약·후반 폭발).
- 노말 무기 = 상성 안 타는 보험. 물/불/풀 무기 = 상성 터지면 폭발딜, 역상성이면 찔끔.

### 무기 슬롯 · 성장 속도 (뱀서 참고)
- **슬롯 상한: 무기 6칸 + 패시브 4~6칸** (뱀서와 동일 원리). 다 못 먹으니 *"뭘 커밋할까"*가 진짜 선택이 되고, 맥스/진화가 의미를 가짐.
- **레벨업 속도**: 처치 XP 기반, 초반 ~25초·후반 ~45초 간격 → 런당 **~50-60회**.
- 무기 6칸 맥스(각 Lv8) ≈ 48회 투자 → **후반부에 풀세팅 완성**. 곡선: 스테이지 1-2 약함(투자) → 3-4 스파이크 → 5-최종 맥스·진화(양학).

### 무기 특색 (설계 1순위 원칙)
스탯만 다른 무기 = 금지. **거동이 다르면 플레이 자세가 달라야** 함:
- 유도 = 방치형 안정 / 일자 관통 = 조준 위치잡기 / 방사 = 근접 파고들기 / 설치 = 함정·유인 / 궤도 = 근접 방어+공격.
- AI 콘텐츠 생성 시 제약: *거동·역할이 겹치지 않게* + 각 무기에 한 줄 "정체성"을 부여.

## 드래프트 (선택)
- 레벨업마다 **랜덤 3택**: 새 무기 / 보유 무기 레벨업 / 패시브.
- 스킵·리롤 카드 소량.
- 스테이지 테마를 알기에 **정보 기반 선택** 가능(운빨 완화).

## 런 구조 (~30분, 6 스테이지)
- **6 스테이지**, 스테이지당 ~5분. **앞 5개 = 랜덤 테마**(로그라이크 변화), **6번째 = 고정 최종**.
- 각 스테이지 = 테마 타입 ~70% + 나머지 혼합 ~30%. **테마는 시작 시 고지** → 정보 기반 선택.
- 스테이지 내부 리듬(비행슈팅 전통):
  - 일반 웨이브 → **중간 보스 1~2회**(짧은 미니 보스) → **위기 세션**(새떼처럼 우르르 몰리는 폭발 인텐시티 구간) → **스테이지 보스**(복합 타입).
- 사이사이 레벨업 3택 드래프트. 바운드 아레나(스테이지별 스킨), 자동수집.
- 예시 랜덤 테마 풀: 바다(물) / 숲(풀) / 화산(불) / 평원(노말) / 늪(물+풀) …
- **최종 스테이지(고정)**: 4속성이 **중구난방으로 섞여** 몰아침 + 고유 연출(특색). 앞선 랜덤 5스테이지로 쌓은 **타입 커버리지의 졸업 시험**.

### 스테이지 예시 — ① 바다
- 적: 물 해파리(다수) + 풀 해초병(소수) + 노말 잡어(소수).
- 정보 기반 선택: **풀 무기**(물에 ×2) 있으면 양학. 불 무기만 있으면 물에 ×0.5라 고전 → 노말/풀로 보강.
- 관대 탄막: 해파리가 느린 물방울 방사(텔레그래프 확실).

### 복합 보스 예시 — "강철 가오리(Manta)"
- 부위별 타입: **엔진=불**(→ 물 ×2) / **좌우 날개=물**(→ 풀 ×2) / **몸체=노말**(무상성, 탱커).
- 전술: 풀 무기로 **날개** 부수면 이동 느려짐 → 물 무기로 **엔진** 부수면 공격 약해짐 → **몸체** 마무리.
- 커버리지 시험대: 물·풀 둘 다 없으면 오래 걸림 = "타입 갖추라"는 허들.
- 관대 패턴: 큰 부채꼴 탄, 텔레그래프 후 발사. 무-트위치.

## 난이도 곡선 (얻은 양학, ~30분)
- **스테이지 1-2 (0-10분)**: 무기 저레벨, 조심 (생존·투자 구간).
- **스테이지 3-4 (10-20분)**: 타입 맞으면 웨이브 녹기 시작 (파워 스파이크).
- **스테이지 5-최종 (20-30분)**: 커버리지 완성 + 무기 맥스·진화 → 양학. 단 중간 보스·위기 세션·복합 보스가 커버리지를 시험.
- 스테이지 내부: 웨이브 → 중간 보스 → 위기 세션(새떼) → 스테이지 보스의 완급.
- 실패 원인 = 타입 구멍 방치 / 몰빵 타이밍 실패 / 위기 세션 대응 실패.

## AI 콘텐츠 파이프라인 (빌드타임, Path A)
- **생성(AI)**: 타입표 · 무기표(거동×타입×진화) · 적 생태(스테이지별 구성) · 복합 보스 부위 스크립트 · 탄막 패턴 · 드래프트 풀 · 난이도 스케줄 → 정적 JSON.
- **시뮬(헤드리스 봇)**: 다양한 드래프트 정책으로 수천~만 판 → 수치 증명: *어떤 스테이지·운(RNG)도 클리어 가능(막다른 운 없음) / 특정 무기·타입이 지배적이지 않음 / 난이도가 하드벗페어 창에 안착*.
- **산출물** = 공모전 "AI 활용 기술 문서"의 근거.

## 미결 / 옵션
- **타입 융합(듀얼/합성)** → 2-3주차 옵션(정체성·깊이 강화, 콘텐츠 부담↑).
- 무기 개수·스테이지 수·정확한 수치 = 프로토타입에서 튜닝.
- 제목 미정.

## 1주차 프로토타입 목표 (손맛 검증)
바운드 아레나 + 이동 + 무기 2~3개(거동×타입) + 적 2타입 + **타입 상성 데미지** + 레벨업 3택 → *"타입 맞춰서 녹이는"* 손맛이 실제로 재밌는지 검증. (콘텐츠·AI 파이프라인 이전, 하드코딩 데이터로.)
