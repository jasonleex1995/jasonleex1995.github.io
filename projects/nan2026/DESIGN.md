# NAN 2026 게임 설계 — 한 판 스펙 v0.5

> 제목: **미정** (가제: *TETRA*). 살아있는 스펙이며 진행하며 갱신합니다.
> 시간순 논의·결정 기록은 [log.md](log.md) 참고.

## 한 줄 컨셉
**뱀서식 무기 아스널 × 물·불·풀·노말 속성 스탠스 × 비행 슈팅.** 랜덤 테마 스테이지를 돌며 속성을 맞춰 갖춰 양학하는 ~30분 오락실 로그라이트.

## 설계 기둥 (Design Pillars)
- **능동 공격** — 자동발사, 유저는 이동·위치 + **속성 스탠스 전환**. 여러 무기 동시 발사(뱀서식).
- **선택이 만든 양학** — 속성 투자 + 무기 맥스 투자로 얻어내는 지배.
- **정보 기반 선택 > 운빨** — 테마로 적 속성 예측 + **리롤/상점**으로 운 의존 완화.
- **적당한 관대 숙련** — 느리고 큰 텔레그래프 탄막, 무-트위치. 압박은 반응이 아니라 위치·판단.
- **의미 있는 이동** — 알맞은 무기를 알맞은 적/부위에. 바운드 아레나, 무-노가다(자동수집).
- **오락실 감성** — 세이브/계정 없음. 한 판 완결 + 점수. (로컬 최고점수만 localStorage.)
- **무API·오프라인·저용량 바닐라** — AI는 빌드타임에 콘텐츠 생성 + 시뮬 밸런싱.

## 속성 시스템 ★핵심★

### 상성 (4종)
| 공격 ↓ \ 방어 → | 물 | 불 | 풀 | 노말 |
|---|---|---|---|---|
| **물** | ×1 | **×2** | ×0.5 | ×1 |
| **불** | ×0.5 | ×1 | **×2** | ×1 |
| **풀** | **×2** | ×0.5 | ×1 | ×1 |
| **노말** | ×1 | ×1 | ×1 | ×1 |

- 물 > 불 > 풀 > 물. **노말**은 항상 ×1(안정적 축).
- **적의 공격에는 속성이 없음** (확정). 적 탄은 그냥 탄 → **내 컨트롤로 회피**. 속성은 *내가 때릴 때만* 작동.

### 무기는 고정, 속성은 투자 + 스탠스 전환 ★
- **무기 = 거동만 고정** (유도/관통 등). 무기 자체에 속성이 박혀있지 않음.
- **속성은 별도 투자 축**: 드래프트로 `불 +1`, `물 +1` … 누적. 상한 = 무기 슬롯 수(4).
- **스탠스 전환**: 키로 현재 속성을 바꾸면, **투자 레벨 N만큼 슬롯 순서대로 앞의 N개 무기가 그 속성으로 발사**. 나머지는 **무속성(×1)**.
  - 예) 무기 4개, 불+2·물+1·풀+0 → **불 스탠스**: 1~2번 무기 = 불, 3~4번 = 무속성. **물 스탠스**: 1번 = 물, 2~4번 = 무속성. **풀 스탠스**: 투자 0이라 전부 무속성.
- **슬롯 재정렬 가능** (레벨업/일시정지 화면에서 드래그) → 로직은 "순서대로 N개"로 깔끔하고 agency도 확보.
- **전환 쿨다운 없음(즉시)** — 이미 "투자한 만큼만" 이 게이트. 보스 부위별 대응이 답답하지 않게.
- **키 = 고정 할당**: `1`=무속성 `2`=불 `3`=물 `4`=풀 (또는 QWER). 머슬메모리 위해 런마다 안 바뀜.
- **왜 좋은가**: 무기 정체성 유지 + 속성이 독립 성장축(드래프트 다양성↑) + **"항상 최적 매칭" 붕괴 방지**(전환은 자유지만 투자한 만큼만) + **특화↔분산 다이얼**(불+4 = 전환 시 4개 전부 불 = 대양학, 대신 타 속성 스테이지에선 전부 무속성) + **노말이 안전 바닥**(절대 브릭 없음) + 전환이 화면에 보임.
- **복합 보스와 시너지**: 엔진=불 → **물 스탠스**, 날개=물 → **풀 스탠스**, 몸체=노말 → 아무거나. 보스전이 살아있는 스탠스 퍼즐.

## 적 공격 거동 · 상태이상
속성은 없지만 **거동은 다양**하게 — 플레이어 무기용 **탄 엔진을 그대로 재사용**(저비용).

| 적 공격 거동 | 주는 압박 |
|---|---|
| 직선탄 / 확산탄 | 기본 위빙 |
| **유도탄** | 계속 움직이게 강제 |
| **레이저**(차지 후 발사) | 텔레그래프 읽고 라인 회피 |
| 설치 장판 | 공간 제한 |
| **상태이상 탄** | ↓ |

**상태이상** (관대함 유지가 관건):
- **둔화**(이동속도 ↓, 수 초) — 위치 압박은 주되 즉사 아님. 기본 채택.
- **짧은 스턴**(0.5~1초) — 강력 → **큰 텔레그래프 + 드물게**, 또는 Hard 이상에서만 등장.
- 대응: 코인 유틸 **상태이상 저항/해제**(비-빌드라 코인 규칙에 부합).

> 패시브 후보: **피격 후 일시 ‘투명’** — 데미지 무효가 아니라 **적의 조준·유도 타겟에서 제외**(무적시간과 다른 별개 메커닉, 유도탄 많은 구간에서 빛남).

## 조작 (양손)
- **이동**: 방향키(오른손). 자동발사.
- **속성 스탠스**: `1234` 또는 `QWER`(왼손).
- 반사신경이 아니라 **판단** 중심 → 무-트위치 유지.

## 무기 (거동 × 진화)
무기는 **거동 패밀리**로 정의(속성은 스탠스가 부여). 각 무기 Lv1~8, **맥스 시 진화**(상위 폼) — 초반 약하지만 참고 키우면 폭발 = 얻은 양학.

**거동 12 패밀리** (조사 기반):
| 패밀리 | 어떻게 나가나 | 플레이 자세 |
|---|---|---|
| 포워드/벌컨 | 정면 연사, 레벨업 시 얕은 콘 | 코앞에서 밀어붙이기 |
| 확산/팬 | 3~5발 부채꼴 | 레인 제압, 측면 위치 |
| 유도/시커 | 최근접 자동추적 | 저부담 칩딜(관대 앵커) |
| 관통 랜스/빔 | 좁고 강한 관통선 | **줄 세우기**, 대보스 저격 |
| 오빗/퍼리미터 | 몸 주위 공전체 | **파고들어 몸빵** |
| 오라/필드 | 자기 중심 지속 장판 | 근접 배짱, 위기세션 학살 |
| 설치존/기뢰 | 바닥에 장판·지뢰 | **흐름 예측**, 유인 |
| 부메랑/리턴 | 나갔다 돌아오며 2히트 | 라인 정렬 |
| 포격/랜덤타격 | 주기적 랜덤 강타 | 방치형 커버 |
| 후방/전방위 | 뒤·옆 사각 커버 | 측후방 안전, 기동 |
| 드론/옵션 | 위성 유닛이 독자 사격 | **화력 배치**, 사각 저격 |
| 노바/화면청소 | 주기적 대형 폭발 | 위기세션 해답, 캡스톤 |

> **특색 원칙 (설계 1순위)**: 스탯만 다른 무기 금지. 거동이 다르면 **플레이 자세**가 달라야 함. AI 생성 시 *거동·역할 겹침 금지* 제약.

## 슬롯 · 드래프트 · 성장
- **무기 4칸** (6은 자동발사 탄으로 화면이 죽 → 적 탄 못 읽음 = 기둥 붕괴).
- **패시브 5~6칸** + 풀도 다양하게 (패시브는 화면을 어지럽히지 않으므로 넉넉히).
- **드래프트 4종류** (레벨업마다 랜덤 3택): **새 무기 / 무기 레벨업 / 속성 레벨업 / 패시브**
- **레벨업 ~50-60+회/런** → 성장 느낌 극대화. 전부 못 찍으므로 선택이 의미를 가짐.
- 곡선: 스테이지 1-2 약함(투자) → 3-4 스파이크 → 5-최종 맥스·진화(양학).

## 런 구조 (~30분, 6 스테이지)
- **6 스테이지**, 스테이지당 ≈5분 = **잡몹 페이즈 ~2분 + 보스 페이즈 최대 3분**. 6 × 5 ≈ 30분(Normal 기준).
- **잡몹 페이즈(~2분)**: 스크롤되며 적이 지나감. **안 잡아도 되지만 안 잡으면 XP를 못 얻음** → *얼마나 공격적으로 파밍할지*가 리스크/리워드 선택. 중간 보스도 선택적(큰 XP).
- **보스 페이즈: 게임시간 3분 타이머** — 못 잡으면 사망. **파밍+빌드의 DPS 시험지** = "잘 골랐나"를 검증하는 허들.
- **앞 5개 = 랜덤 테마**(바다·숲·화산·평원·늪 등), **6번째 = 고정 최종**(4속성 중구난방 + 고유 연출 = 커버리지 졸업 시험).
- 각 스테이지 = 테마 속성 ~70% + 혼합 ~30%. **테마는 시작 시 고지** → 정보 기반 선택.
- 스테이지 내부 리듬(비행슈팅 전통): 웨이브 → **중간 보스 1~2회** → **위기 세션**(새떼) → **스테이지 보스**(복합 타입).
- 바운드 아레나(스테이지별 스킨), 자동수집.

### 복합 보스 예시 — "강철 가오리(Manta)"
- 부위별 속성: **엔진=불**(→ 물 스탠스) / **좌우 날개=물**(→ 풀 스탠스) / **몸체=노말**(무상성 탱커).
- 전술: 풀로 날개 부수면 이동 느려짐 → 물로 엔진 부수면 공격 약해짐 → 몸체 마무리.
- 관대 패턴: 큰 부채꼴 탄, 텔레그래프 후 발사. 무-트위치.

## 난이도 = 배속 (스피드런) ★
**게임 클럭 하나만 곱하면** 보스 타이머·런 길이가 자동으로 따라옴(변수 1개로 난이도 전체 스케일 — 구현 극저비용).
모든 것(적 탄·내 발사·이동·스폰)이 동일 배속. 보스 타이머는 항상 **게임시간 3분** → 실제 = `3분 ÷ 배속`.

| 난이도 | 배속 | 보스 타이머(실제) | 런 길이(실제) | **점수 배율** |
|---|---|---|---|---|
| **Normal** | ×1.0 | 3:00 | ~30분 | ×1.0 |
| **Hard** | ×1.5 | 2:00 | ~20분 | ×1.6 |
| **Hell** | ×2.0 | 1:30 | ~15분 | ×2.5 |
| **Disaster** | ×3.0 | 1:00 | ~10분 | ×4.0 |

- 점수 배율은 **초선형** → 리스크 감수의 이유.
- 기본(Normal)은 관대 유지 → **난이도는 옵트인**(무-트위치 기둥 보존). 짧게 하고 싶으면 Hell(15분).
- 로컬 최고점수는 **난이도별**로 기록.

## 재화(코인) · 상점
> **규칙: 코인으로 살 수 있는 건 선택권·보험·비-빌드 유틸뿐. 무기/속성 레벨은 절대 판매 금지** (빌드는 레벨업의 영역 — 코인이 침범하면 성장의 의미가 무너짐).

| 분류 | 항목 | 비고 |
|---|---|---|
| **선택권**(운 완화) | **리롤** | 살 때마다 가격 ↑ |
| **생존/보험** | **회복 물약** | 구매할수록 가격 ↑ |
| | **폭탄 스톡** | ↓ 아래 상세 — *살아남는 카드*이지 딜 카드가 아님 |
| | **실드**(1회 피격 무효) | 명확·가시적. (‘무적시간 ↑’은 체감 약해 폐기) |
| | **컨티뉴** | **런당 1회 하드캡** + 비쌈 + 퍼펙트/무피격 보너스 상실 |
| | **시간 연장 토큰**(소비형) | 상점서 구매 → **보스전 중 직접 발동**(타이머 빨간불일 때) |
| **비-빌드 유틸** | 방어력 ↑ / 최대체력 소폭 ↑ | |
| | 이동 속도 ↑ | |
| | **코인·XP 자동수집 반경 ↑** | ‘줍기 노가다’ 제거와 일치 |
| | **상태이상 저항/해제** | 적 상태이상 탄 대응 |
| **금지** | ~~무기 레벨~~ / ~~속성 레벨~~ | 레벨업의 영역 |
| | ~~밴시~~ | 운을 과하게 제거 → 로그라이트 긴장 소멸 (폐기) |

### 폭탄 설계 (슈팅 전통)
- **잡몹**: 화면 광역 즉사급 + **적 탄 소거** ← 폭탄의 진짜 정체성(위기 탈출).
- **보스/중간보스**: **최대 체력의 3~5%만** + 상한 → 폭탄 스팸으로 보스를 녹이는 것 원천 차단.
- → 폭탄 = *살아남는 카드*. 보스 3분 타이머의 긴장이 유지됨.

**희소성 설계** (항상 살짝 부족하게):
- 코인은 **엘리트·보스·일부 잡몹만** 드랍.
- 리롤·물약 **가격 상승**.
- ★ **남은 코인 = 런 종료 시 점수로 환산** → 쓰면 점수를 잃음. 모든 구매가 진짜 기회비용이 되고 *"아껴서 점수 vs 써서 생존"*의 오락실다운 긴장 발생.

## 회복
- **레벨업 드래프트에는 넣지 않음** (‘회복 vs 성장’ feel-bad 방지).
- **스테이지 사이 부분 회복** + **상점 구매(가격 상승)** + 드물게 엘리트 드랍.

## 점수 (오락실)
| 항목 | 의도 |
|---|---|
| 처치 점수 | 기본 |
| **상성 처치 보너스**(×2로 처치 시 추가) | 속성 시스템을 점수로 보상 |
| **남은 시간 보너스**(보스 타이머 잔여) | 빠른 격파 = 좋은 빌드 보상 |
| 무피격 / 보스 격파 보너스 | 숙련 보상 |
| **남은 코인 환산** | 절약 ↔ 생존의 기회비용 |
| **× 난이도 배율** | 배속 리스크 보상 |
- 세이브·계정 없음. **로컬 최고점수만 localStorage**(난이도별) — "이 기계의 최고 기록" 감성.

## AI 콘텐츠 파이프라인 (빌드타임, Path A)
- **생성(AI)**: 상성표 · 무기 거동표(12패밀리 × 진화) · 속성 투자 곡선 · 적 생태(스테이지별 구성) · 복합 보스 부위 스크립트 · 탄막 패턴 · 드래프트 풀 · 상점 가격 · 난이도/시간제한 스케줄 → 정적 JSON.
- **시뮬(헤드리스 봇)**: 다양한 드래프트 정책으로 수천~만 판 → 수치 증명: *어떤 운(RNG)이어도 각 보스를 3분 타이머 내 격파 가능(막다른 운 없음) / 특정 무기·속성이 지배적이지 않음 / 난이도가 하드벗페어 창에 안착 / 코인 수급이 항상 살짝 부족*.
- **산출물** = 공모전 "AI 활용 기술 문서"의 근거.

## 미결 / 옵션
- 속성 스탠스 키: `1234` vs `QWER`.
- 제목 미정.
- 무기·패시브 최종 개수, 코인 수급/가격, 난이도 배율 수치 = 프로토타입 + AI 시뮬로 튜닝.

## 1주차 프로토타입 목표 (손맛 검증)
바운드 아레나 + 이동 + 무기 2~3개(거동) + **속성 스탠스 전환(1234)** + 적 2~3속성 + **상성 데미지** + 레벨업 3택 → *"스탠스 바꿔서 상성으로 녹이는"* 손맛이 실제로 재밌는지 검증. (콘텐츠·AI·상점·점수는 그 다음, 하드코딩 데이터로.)
