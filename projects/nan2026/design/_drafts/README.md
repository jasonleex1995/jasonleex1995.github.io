# ⚠️ 원자재(raw drafts) — 설계 스펙 아님

이 폴더의 6개 문서는 갭 분석 후 **병렬로 독립 작성**된 초안이다.
개별적으로는 유용한 재료지만, **서로 모순된다** (핵심 상수·스키마·검증 게이트를 각자 독립적으로 확정함).

**이 문서들을 그대로 개발 근거로 쓰면 안 된다.**
정본(canon) 확정 → 정본 기준 재작성 → 검증을 거친 뒤에야 스펙이 된다.

주요 모순: 방어력 모델 3종 / 텔레그래프 하한 3종(모두 하드 게이트) / 데이터 파일 매니페스트 6종 /
HP 스케일 100 vs ~10 / 보스 HP 구조 2종 / 시뮬 인증 게이트 상호 만족 불가 / 팔레트 2종 / 아레나·히트박스 불일치 등.
